## 二、实验内容
1. **反走样（抗锯齿）**
   - 使用亚像素精度浮点坐标，不直接截断为整数
   - 对 3×3 邻域像素按距离分配颜色权重
   - 实现平滑边缘，消除阶梯状走样

2. **贝塞尔曲线（Bézier Curve）**
   - 全局控制：修改任意控制点，整条曲线都会变化
   - 阶数 = 控制点数量 - 1
   - 采用德卡斯特里奥算法计算

3. **均匀三次B样条曲线（B-Spline）**
   - 局部控制：修改一个控制点只影响附近小段曲线
   - 固定三次多项式，可无限增加控制点
   - 自动平滑拼接，无高阶振荡问题
   - n 个控制点生成 n-3 段曲线

## 三、操作说明
| 操作 | 功能 |
|------|------|
| 鼠标左键 | 添加控制点 |
| B 键 | 切换 **贝塞尔 / B样条** 模式 |
| C 键 | 清空所有控制点 |
| 关闭窗口 | 退出程序 |

## 四、实验效果
### 1. 反走样效果
- 曲线边缘平滑无锯齿
- 对比传统光栅化，视觉质量显著提升

### 2. 贝塞尔曲线 vs B样条曲线
- **贝塞尔**：全局控制、紧贴控制点、高阶易波动
- **B样条**：局部控制、更平滑、数值稳定、适合大量控制点

1. 亚像素级反走样渲染算法，有效消除曲线走样
